# Boutique Website

Welcome to Style Hive, a boutique fashion website designed to showcase elegant collections with a modern, professional look.  
This project was built as part of an internship to demonstrate real-world web development skills, focusing on design, branding, and deployment.

##  Features
- **Responsive Design**: Works seamlessly across desktop and mobile devices.
- **Elegant Theme**: Dark brown background with gold accents for a premium boutique feel.
- **Hero Section**: Eye-catching banner with a tagline and call-to-action button.
- **Product Gallery**: Showcases dresses, bags, accessories, and footwear with names and descriptions.
